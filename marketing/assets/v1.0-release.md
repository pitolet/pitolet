# Pitolet v1.0

Pitolet is a web-native design tool for developers and coding agents. The
canvas renders real DOM and CSS, design files are readable JSON in your repo,
and the same document exports deterministically to React + Tailwind or plain
HTML + CSS.

## Highlights

- Real flexbox, grid, breakpoints, and interaction states on the canvas.
- Read/write MCP tools for frames, nodes, tokens, comments, screenshots,
  project export, and drift checking.
- Live, validated agent edits that remain undoable in the editor.
- Design tokens, components, variants, comments, preview mode, and code panel.
- Git-native `.pitolet.json` documents with hot reload from disk.
- Self-hosting through npm or Docker, plus a hosted cloud edition with
  workspaces, scoped agent tokens, share links, and version history.

## Try it

```sh
npx pitolet
```

Then open `http://localhost:4517` and connect an MCP client to
`http://localhost:4517/mcp`.

## Known gaps

- Human multiplayer cursors are not implemented yet.
- Advanced grid editing is still thinner than flexbox editing.
- Nested component instances remain limited.

See the [README](https://github.com/pitolet/pitolet#readme) for Docker,
development, and MCP setup instructions.
